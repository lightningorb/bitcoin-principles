# Svelte Fedi Mod

This is a very tiny svelte frontend of the fedi mod.

## Useful links

https://github.com/fedibtc/fedi-mods

## Recommended versions

`node v18.18.2`

`npm 9.8.1`

## Installing dependencies

`npm install`

## Building

`npm run build`

## Running

`npm run dev`

## Deploying

For deployment, speak with Orb for access keys etc.

`npm run deploy`
